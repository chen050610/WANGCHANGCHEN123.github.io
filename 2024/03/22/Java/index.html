Java的基础语法

public class Hello {
    public static void main(String[] args) {
        System.out.println("你好");
    }
}

  • 主函数 & 程序的入口

  • 文件名

    一个文件中只能有一个public的类，并且文件名必须和public的类名保持一致
如果文件中有多个类，文件名与public的类名保持一致
如果文件中有多个类 且无public类，文件名抗日一是任意的类

  • 类名 首字母大写，驼峰的命名

  • 类修饰符 public,default

  • 类中的成员修饰符：public,private,protected,default

  • 静态成员：无需实例化就可以直接调用

    class Person  {
    public static void  f1(){
        //static静态的方法，无需实例就可以调用，也就只可以类调用
        System.out.println("111");
    }
}

public class Hello {
    public static void main(String[] args) {
        System.out.println("你好");
        Person.f1();//111
    }
}

    • 实例方法：需要实例以后在调用

      class Person  {
    public  void  f1(){
        System.out.println("111");
    }
}

public class Hello {
    public static void main(String[] args) {
        System.out.println("你好");
        Person person = new Person();//实例对象的写法是这样的
        person.f1();//对象调用
    }
}

  • void代表方法没有返回值

        public static void main(String[] args) {
        System.out.println("你好");
        Person person = new Person();//实例对象的写法是这样的

public class Hello {

输入和输出

输入需要外部的包，

public class Hello {
    public static void main(String[] args) {
        //引入的Scanner相当于一个类，System.in感觉时使用系统命令获取输入
        Scanner input  = new Scanner(System.in);
        String text  =  input.nextLine();
        //输出
        System.out.println(text);
    }
}

输出语句的不同

        //输出
        System.out.println(text);//输出最后会带换行
        System.out.print(text);//输出最后不带换行

条件语句

public class Hello {
    public static void main(String[] args) {
        System.out.print("");

            System.out.println("少年");
        } else if (age>= 18 && age < 24){
            System.out.println("青年");
        } else {
            System.out.println("老年");
        }
    }
}

do while循环

至少执行一次

public class Hello {
    public static void main(String[] args) {
        int count = 0 ;
        do {
            System.out.println(count);
            count++;
        } while (count < 3);
    }
}
