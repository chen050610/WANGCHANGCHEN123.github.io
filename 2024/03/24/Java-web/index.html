Java web

  • web开发

    • web,网页的意思，

    • 静态web

      • 就是没有服务器的熏染，数据是写死的

    • 动态的web

      • 技术栈：Servlet/JSP，ASP，d

在Java中动态资源开发技术统称为Java web

web应用程序

web应用程序：可以提供浏览器访问的程序

静态页面

  • 存在的缺点。web无法动态的更新所有的用户看见的都是一个页面

  • 他无法和数据库交互，数据无法持久化，用户无法交互

动态页面

  • 页面会动态的展示，因人而异

  • 缺点：加入服务器的动态web资源出现错误，我们需要重新编写我们的后台程序

    • 停机维护

  • 优点：web可以的动态的更新，数据的持久化（数据的储存和删除）

web服务器

  • ASP：

    • 微软：国内最早流行的，在HTML中嵌入VB的脚本，ASP+COM;
    • 在ASP开发中，基本一个页面有很多的业务代码
    • 维护的成本高
    • 使用的语言是C#

  • PHP：PHP开发的速度很快，功能很强大，跨平台，代码简单（70% WP）

    • 无法承载大访问量的情况（局限性）

  • JSP/Servlet:

    • sun公司 主推的 B/S架构，基于Java语言，所有的大公司或火星和一些开源的组件，都是用Java写的，而且加可以承载三高的问题；语法像ASP
    • B/S:浏览器和服务器
    • C/S:客户端和服务器

web服务器

用来接收用户的请求，负责给用户响应

  • Tomcat

  • IIS
