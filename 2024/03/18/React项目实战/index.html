懒加载

路由表的懒加载的实现方法，对于懒加载组件需要在外面的一层套上Suspense

import { lazy , Suspense } from 'react'
import {Navigate} from 'react-router-dom'

const About = lazy(()=>import('@/views/About.tsx'))
const Home = lazy(()=>import('@/views/Home.tsx'))



const routes = [
    {path:'/',element:<Suspense fallback={<h1>loading</h1>}><Navigate to='/home'/></Suspense>},
    {path:'/home',element:<Suspense fallback={<h1>loading</h1>}><Home/></Suspense>},
    {path:'/about',element:<Suspense fallback={<h1>loading</h1>}><About/></Suspense>},
]
export default routes

简写的方法，使用函数进行封装

const About = lazy(()=>import('@/views/About.tsx'))
const Home = lazy(()=>import('@/views/Home.tsx'))

const withLoadingComponent = (comp:JSX.Element)=>{
    return (
        <Suspense fallback={<h1>loading</h1>}>{comp}</Suspense>
    )
}

const routes = [
    {path:'/',element:<Suspense fallback={<h1>loading</h1>}><Navigate to='/home'/></Suspense>},
    {path:'/home',element:withLoadingComponent(<Home/>)},
    {path:'/about',element:withLoadingComponent(<About/>)},
]
export default routes

js中字符串的一些方法

JavaScript中字符串有很多方法可以用来操作和处理，下面列举一些常用的字符串方法：

  1. length: 返回字符串的长度。
javascriptCopy Codeconst str = "Hello, World!";
console.log(str.length); // 输出 13
  1. charAt(index): 返回指定索引位置的字符。
javascriptCopy Codeconst str = "Hello";
console.log(str.charAt(1)); // 输出 e
  1. substring(startIndex, endIndex): 返回指定索引范围内的子串。
javascriptCopy Codeconst str = "Hello, World!";
console.log(str.substring(1, 5)); // 输出 ello
  1. indexOf(substring): 返回子串第一次出现的索引，如果未找到则返回 -1。
javascriptCopy Codeconst str = "Hello, World!";
console.log(str.indexOf("World")); // 输出 7
  1. replace(oldValue, newValue): 替换字符串中的子串。
javascriptCopy Codeconst str = "Hello, World!";
console.log(str.replace("Hello", "Hi")); // 输出 Hi, World!
  1. toUpperCase(): 将字符串转换为大写。
javascriptCopy Codeconst str = "hello";
console.log(str.toUpperCase()); // 输出 HELLO
  1. toLowerCase(): 将字符串转换为小写。
javascriptCopy Codeconst str = "WORLD";
console.log(str.toLowerCase()); // 输出 world
  1. trim(): 去除字符串两端的空格。
javascriptCopy Codeconst str = "   Hello, World!   ";
console.log(str.trim()); // 输出 Hello, World!

  1. split(separator, limit): 这是最常见的字符串分割方法，它可以根据指定的分隔符将字符串分割成子串，并返回一个由这些子串组成的数组。可选的 limit 参数用于限制返回的子串数量。

    javascriptCopy Codeconst str = "apple,banana,orange";
const arr = str.split(",");
console.log(arr);  // 输出 ["apple", "banana", "orange"]

  2. substring(startIndex, endIndex): 这个方法不是真正意义上的字符串分割方法，但它可以用来获取指定范围内的子串。这个方法并不会改变原始字符串，而是返回一个新的子串。

    javascriptCopy Codeconst str = "Hello, World!";
const sub = str.substring(0, 5);
console.log(sub);  // 输出 Hello

  3. slice(startIndex, endIndex): 类似于 substring() 方法，slice() 也可以用于获取指定范围内的子串，并返回一个新的子串。不同之处在于，slice() 方法允许使用负数作为索引，表示从末尾开始计算索引位置。

    javascriptCopy Codeconst str = "Hello, World!";
const sub = str.slice(7);
console.log(sub);  // 输出 World!
